cbrncl:RadDetectorKindCodeSimpleType

simpleType RadDetectorKindCodeSimpleType in namespace http://release.niem.gov/niem/codes/cbrncl/4.0/

Definition

A data type that defines the various code values for the kinds of radiation detectors that may compose a radiation instrument.
