cbrn:RadRawTotalExposureValue

element RadRawTotalExposureValue in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A TotalExposure measurement data element(s) used to produce derived data. There shall be no duplicate IDREF values in the list. This is required whenever the element is used within a DerivedData block, and prohibited otherwise.
