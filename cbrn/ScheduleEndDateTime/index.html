cbrn:ScheduleEndDateTime

element ScheduleEndDateTime in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A date when the schedule is no longer effective. If not provided, then the end date is considered to be indefinite. The end date, if provided, must be later than the start date.
