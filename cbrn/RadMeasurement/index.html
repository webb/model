cbrn:RadMeasurement

element RadMeasurement in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A set of data for a radiation measurement at a particular StartDateTime, for a RealTimeDuration, of a particular MeasurementClassCode that consists of readings from any number of one or more of the following: - a radiation detector - an occupancy sensor - a positioning sensor that captures the location of a radiation measurement instrument, radiation detector, or measured item - the state of a radiation measurement instrument, radiation detector, or measured item
