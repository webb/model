cbrn:RadItemMeasurementGeometryDescriptionText

element RadItemMeasurementGeometryDescriptionText in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A description of the position and/or shape of the geometry used in the measurement of this item; e.g., shape of the item, item orientation relative to the radiation detectors, position of any attenuators used.
