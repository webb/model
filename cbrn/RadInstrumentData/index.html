cbrn:RadInstrumentData

element RadInstrumentData in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A set of all the measurement and analysis data reported by a radiation detection instrument, and all the reported information on the instrument, its radiation detector(s), and the item(s) it measured.
