hs:JuvenileChargeStatusOffenseIndicator

element JuvenileChargeStatusOffenseIndicator in namespace http://release.niem.gov/niem/domains/humanServices/4.1/

Definition

True if the alleged act is an offense when committed by a juvenile but not when committed by an adult; false otherwise.
