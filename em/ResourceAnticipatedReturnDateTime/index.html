em:ResourceAnticipatedReturnDateTime

element ResourceAnticipatedReturnDateTime in namespace http://release.niem.gov/niem/domains/emergencyManagement/4.1/

Definition

A date when the resource is expected to be returned or DEMOBed. Completed for a Request Resource. This element with the Requested Arrival Date / Time provides the estimated duration of resource deployment
