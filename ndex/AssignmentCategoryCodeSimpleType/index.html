ndex:AssignmentCategoryCodeSimpleType

simpleType AssignmentCategoryCodeSimpleType in namespace http://release.niem.gov/niem/codes/fbi_ndex/4.1/

Definition

A data type for a code that identifies a subject's assignment category.
