ndex:IntellectualPropertySubjectMatterCategoryCodeSimpleType

simpleType IntellectualPropertySubjectMatterCategoryCodeSimpleType in namespace http://release.niem.gov/niem/codes/fbi_ndex/4.1/

Definition

A data type for a code that identifies the type of subject matter protected by intellectual property rights.
