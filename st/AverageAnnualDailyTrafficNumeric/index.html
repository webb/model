st:AverageAnnualDailyTrafficNumeric

element AverageAnnualDailyTrafficNumeric in namespace http://release.niem.gov/niem/domains/surfaceTransportation/4.1/

Definition

An AADT value to represent the current data year. For two-way facilities, provide the AADT for both directions; provide the directional AADT if part of a one-way couplet or for one-way streets
