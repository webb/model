scr:ActivityPriorityQuantity

element ActivityPriorityQuantity in namespace http://release.niem.gov/niem/domains/screening/4.1/

Definition

A number of the Priority value of the activity, with 0-4 as gradient normal, 4 as the default, and 5-9 as gradient expedited priority.
