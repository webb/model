im:TotalSponsorFundingSupportAmount

element TotalSponsorFundingSupportAmount in namespace http://release.niem.gov/niem/domains/immigration/4.1/

Definition

A number indicating the funding amount provided by the Program Sponsor to the Exchange Visitor (EV), in order to support his/her participation within a designated EV Program.
