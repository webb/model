im:TwoYearRequirementIndicator

element TwoYearRequirementIndicator in namespace http://release.niem.gov/niem/domains/immigration/4.1/

Definition

True if the Exchange Visitor (EV) is subject to a two year residence requirement as a result of G-1, G-2, or G-3 program participation and funding receipt from current program sponsor; false otherwise.
