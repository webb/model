j:IntoxicationIntoxicantIllegalIndicator

element IntoxicationIntoxicantIllegalIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

True if a substance that contributed to the intoxication of a person was illegal at the time and location of the event; false if it was legal.
