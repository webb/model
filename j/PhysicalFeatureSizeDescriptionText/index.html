j:PhysicalFeatureSizeDescriptionText

element PhysicalFeatureSizeDescriptionText in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

A description of the size of a physical feature if not measured in height and width, such as a numeric size for a prosthetic or a tattoo covering a specific area.
