j:ChargeAgencyRecordIdentification

element ChargeAgencyRecordIdentification in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

A case identification assigned by the charging agency. This element, called the Originating Case Agency number (OCA) in the III (Interstate Identification Index), can be used to enter an AFIS or Process Control Number from the arrest fingerprint.
