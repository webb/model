j:ExhibitRemainingProofIndicator

element ExhibitRemainingProofIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

True if an exhibit that has not been admitted as evidence will remain with the court as an offer of proof in the event of an appeal; false otherwise.
