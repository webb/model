j:RapSheetTransactionControlIdentification

element RapSheetTransactionControlIdentification in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

A Transaction Control Number (TCN) is the identification for a fingerprint event, submitted from the Livescan device to AFIS when transmitting fingerprints. The TCN links the offender to the fingerprint event.
