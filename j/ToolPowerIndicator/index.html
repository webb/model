j:ToolPowerIndicator

element ToolPowerIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

True if the tool is operated by a procedure in which manual effort is supplemented or replaced by hydraulic, mechanical or electrical means; false otherwise.
