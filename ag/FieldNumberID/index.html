ag:FieldNumberID

element FieldNumberID in namespace http://release.niem.gov/niem/domains/agriculture/4.0/

Definition

An identifier number assigned to a field of land that is part of a farm tract. Field numbers are assigned by an FSA county office, and are unique within that offics farms. A Field Number is used in conjunction with Tract Number, State FSA Code and County FSA Code for uniqueness. Field number and Tract Number are not necessarily unique based upon State ANSI Code and County ANSI Code in instances where the ANSI codes are not the same as the FSA codes.
