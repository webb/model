ag:CustomerInterAgencyIdentificationID

element CustomerInterAgencyIdentificationID in namespace http://release.niem.gov/niem/domains/agriculture/4.0/

Definition

A system generated, unique internal identifier used between agencies to identify a USDA customer (individual or business). Currently SCIMS Core Customer Identifier is the source value. This will change if a different customer unique identifier is adopted.
