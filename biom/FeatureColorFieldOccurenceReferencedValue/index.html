biom:FeatureColorFieldOccurenceReferencedValue

element FeatureColorFieldOccurenceReferencedValue in namespace http://release.niem.gov/niem/domains/biometrics/4.1/

Definition

A second information item, feature _ field occurrence / FTO, indicates which repeating subfield of the specified field the label is applied to. Note that this is a 1-based index, not a 0-based index. Occurrences are numbered starting with 1.
