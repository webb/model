biom:FeatureColorFieldReferencedCode

element FeatureColorFieldReferencedCode in namespace http://release.niem.gov/niem/domains/biometrics/4.1/

Definition

A data for one or both of the third or fourth information items shall be present in each subfield. The first two information items are mandatory. The first information item, feature - field number / FTF, indicates the field which is annotated with c
