biom:TimeSegmentCommentText

element TimeSegmentCommentText in namespace http://release.niem.gov/niem/domains/biometrics/4.1/

Definition

A comment which allows for comments of any type to be made on a segment, including, but not limited to, word- or phone-level transcriptions, language translations or security classification markings.
