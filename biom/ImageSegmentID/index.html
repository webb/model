biom:ImageSegmentID

element ImageSegmentID in namespace http://release.niem.gov/niem/domains/biometrics/4.1/

Definition

An index identifier for a particular set of segmentation coordinates (e.g., a polygon) which encloses the relevant part of a source image or associated data image.
